# 運用メモ（Agents 向け）

このプロジェクトで動作するエージェント／自動化スクリプトに関する取り決めをまとめます。

1. **返答は原則として日本語で行うこと。**  
   - ログ、標準出力、ドキュメント更新を含め、日本語でのコミュニケーションを基本とする。
   - 国際化が必要な場合は別途注記する。
2. **アーキテクチャ遵守**  
   - Raspberry Pi 側は Lesson 6 (`camera.sh`, `startcam.py`, `picar4.sh`, `webcar.py`) を利用し、Mac 側は MJPEG Pull + Flask への HTTP 制御を実装する。
   - 旧 ESP32-CAM / WebSocket 互換コードは使用しない。
3. **通信設定**  
   - 映像: `http://<PiのIP>:8899/stream.mjpg`
   - 制御: `http://<PiのIP>:5000/?action=command&...`
   - これらのURLが変更された場合は README と本ファイルを更新する。
   - 現在 (2025-11-10) の Pi IP: `192.168.11.4`
4. **ログレベル**  
   - 開発時は DEBUG ログで保存し、運用時は INFO を既定とする。

更新時は変更理由と日付を追記してください。
